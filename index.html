PHP Tool Integration

On one site there exists many good PHP tools such as PHPUnit or PHP CodeSniffer which support the development of PHP scripts. On the other side there powerful IDEs for example PHP Development Tools Project (PDT) but mostly with no possibility to use such tools with them. The Eclipse PHP Tool Integration (PTI) is a collection of Eclipse plugins which aims to close this gap.
